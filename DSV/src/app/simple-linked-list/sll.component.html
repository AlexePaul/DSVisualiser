Lista Simplu Inlantuita

  • O lista simplu inlantuita se bazeaza pe o structura ce contine doua valori(Valoarea de pe pozitia actuala si next, un pointer catre urmatoarea valoare). Capul listei reprezinta prima valoare si este suficienta pentru a parcurge toata lista.

  • Adaugare Numar (la inceput)

    • Complexitate: O(1).
    Adaug numarul la inceputul listei prin a modifica capul listei in noul element apoi modific capul Next al acestuia in fostul prim element.

  • Adaugare Numar (la final)

    • Complexitate: O(n), unde n este numarul de elemente din lista.
    Pentru a putea adauga la finalul listei sunt nevoit sa parcurg intreaga lista deoarece eu tin minte doar capul listei. Dupa ce ajung la finalul listei, actualizez campul Next al ultimului element cu acest nou element

  • Sterge Numar (la inceput)

    • COmplexitate: O(1).
    Pentru a sterge un numar din lista, este suficient sa actualizez capul sa fie egal cu cel de al doilea element.

  • Sterge Numar (la final)

    • Complexitate: O(n), unde n este numarul de elemente din lista.
    Pentru a putea sterge ultimul element trebuie sa parcurg pana la penultimul element, apoi ii setez campul Next al acestuia vid.
