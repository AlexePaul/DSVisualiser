Tabela Hash

  • O tabela hash este o structura de date care stocheaza perechi de cheie-valoare, unde cheile sunt rezultatul aplicarii unei functii de hash valorii. Un exemplu simplu de functie de hash folosita pentru acest scop este modulo. O problema des intalnita in cadrul tabelelor hash este rezultatul identic pentru mai multe valori (dupa aplicarea functiei de hash), pentru a combate asta, fiecare indice din tabel este reprezentat de o lista simplu inaltuita.

  • Cautare numar

    • Complexitate: O(1) in cazul mediu (ideal), O(n) in cel mai rau caz.

  • Inserare numar

    • Complexitate: O(1) in cazul mediu (ideal), O(n) in cel mai rau caz.
    Se calculeaza indicele de tabel folosind functia de hash si se adauga numarul la acest indice.

  • Stergere numar

    • Complexitate: O(1) in cazul mediu (ideal), O(n) in cel mai rau caz.
    Se calculeaza indicele de tabel folosind functia de hash, iar apoi se elimina numarul de la acest indice (in cazul in care sunt mai multe, se parcurge lista pentru a fi gasit).
